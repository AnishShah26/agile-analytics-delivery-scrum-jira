# Agile Analytics Delivery Project | Scrum, Jira & Business Analysis

This is a simulated Agile delivery case study showing how a SaaS churn and revenue-at-risk dashboard can be planned and managed using Scrum, Jira and business analysis artefacts.

The project demonstrates requirements gathering, stakeholder mapping, user stories, acceptance criteria, backlog structuring, sprint planning and delivery documentation for an analytics dashboard.
